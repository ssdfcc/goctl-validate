# goctl-validate
此插件用于生成go-playground/validator的校验器代码。

### 1. 安装 goctl-validate

```
$ GOPROXY=https://goproxy.cn/,direct go install github.com/ssdfcc/goctl-validate@latest
```

### 2. 环境设置
   将 goctl-validate 的路径添加到环境变量。

### 3. 用法
```
$ goctl api plugin -p goctl-go-compact -api user.api -dir .
```
